# csprag-f19-rpn
an RPN Calculator, in Python.  

Has implementation for addition, subtraction, multiplication, division, and exponentiation.

USAGE:  "1 1 +";
        "1 2 + 3 * 4 / 0 ^";
        "3 1 -";
        "4 2 *";
